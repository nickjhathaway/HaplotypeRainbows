# Example sample metadata for [pfIsosHeomeV1](https://nickjhathaway.github.io/HaplotypeRainbows/reference/pfIsosHeomeV1.md)

Per-sample metadata for the samples in
[pfIsosHeomeV1](https://nickjhathaway.github.io/HaplotypeRainbows/reference/pfIsosHeomeV1.md),
for demonstrating the sample-metadata sidebar. Subset to the example
samples with columns that are constant across the subset dropped. The
`sample` column matches `pfIsosHeomeV1`'s `s_Sample`.

## Usage

``` r
pfIsosHeomeV1_sampleMeta
```

## Format

A data frame with 61 rows (one per sample):

- sample:

  Sample identifier (matches `s_Sample`).

- collection_date:

  Collection date, where known.

- country:

  Country of origin.

- site:

  Collection site.

- collection_year:

  Collection year, where known.

- region:

  Geographic region.

- subRegion:

  Geographic sub-region.

- secondaryRegion:

  Broader (continent-level) region.
